Reference projects for Trenz Electronic TE063X modules
Requirements:
Xilinx ISE 13.2 
Xilinx EDK 13.2
Git client

To get projects:
Run Git Bash
git clone git@github.com:Trenz-Electronic/TE063X-Reference-Designs.git
cd TE063X-Reference-Designs
git submodule init
git submodule update
Open TE063X-Reference-Designs/reference-TE0630/system.xmp in Xilinx EDK
